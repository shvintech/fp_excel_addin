# React + Vite

This project is set up using React with Vite, providing a fast development environment with HMR (Hot Module Replacement) and optimized production builds.

## Getting Started

Follow the steps below to set up and run the project.

### Installation

Create a .env file in the project root.

VITE_SUPABASE_URL=https://domain_name.supabase.co/functions/v1

VITE_SUPABASE_PUBLISHABLE_DEFAULT_KEY= BEARER_TOKEN

### Running the Development Server

Start the app with hot reloading:

npm run dev

The project will be available at:

http://localhost:5173

### Building for Production

To create a production-optimized build:

npm run build

### Previewing the Production Build

Preview the built application locally:

npm run preview
